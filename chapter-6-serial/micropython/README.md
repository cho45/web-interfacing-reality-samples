# Chapter 6: MicroPython Web Serial

Web Serial API を使用して、ブラウザから Raspberry Pi Pico 上の MicroPython REPL を操作するサンプルです。

## セットアップ

1. Raspberry Pi Pico に MicroPython ファームウェアをインストールします。
2. Pico を PC に USB 接続します。

## 使い方

1. `index.html` をブラウザで開きます（Web Serial API は Chrome 等の対応ブラウザが必要です）。
2. **「接続」**ボタンをクリックし、シリアルポートを選択します。
3. コマンドの送信や、Lチカ、温度センサーの読み取りなどのデモを実行できます。
